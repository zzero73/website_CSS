一、我的礁島想像

📌 設計理念

為何這樣設計礁島：
我想像中的礁島是一個完全獨立於世外的空間，因為我非常嚮往這種寧靜感，因此設定為只能透過私人小型飛機抵達，也暗示了島嶼的隱密性與專屬性。島上的生態原始且豐富，擁有綿延的沙灘與茂密的叢林。沙灘上的漂流木，則是用來點綴未經雕琢的美感。島上的建築物則刻意選擇傳統手工搭建的風格，並強調其自然材料與風化痕跡，目的在於使其與周遭環境融為一體，展現時間與自然的印記，而非現代化的突兀建築。

生成結果與自己想像的差異：
在生成圖片之前，我對於私人飛機的想像其實是更偏向傳統的浮筒飛機。然而，生成結果中的飛機是帶有輪子的水陸兩用飛機，雖然也能在水上降落，但視覺上與我最初設想的有所不同。儘管如此，這部分的差異並不算太大，仍然符合「私人飛機可以抵達」的核心概念。總體而言，生成結果與我的想像非常接近。礁島的整體景觀、茂密的叢林、綿密的沙灘、風化的建築以及漂流木等元素都呈現得相當到位。

八、（額外專案）Gemini Canvas 智慧投資系統

🚀 專案介紹

這是一個專為大學生設計的互動式理財 Web App。利用 Gemini Canvas 協作開發， 結合 Tailwind CSS 打造極具質感的 Glassmorphism (磨砂玻璃) 介面， 將複雜的投資邏輯轉化為直觀的視覺體驗。

功能亮點：
  • 多元演算模型：內建隨機森林 (RF)、均線策略 (MA) 與價值回歸 (VR) 三種模擬引擎。
  • 客製化參數：針對小資族設定 (1k-40k 預算)，並可切換長/短期投資視角。
  • 動態視覺回饋：包含載入動畫、適配度計算與模擬趨勢圖表的即時渲染。
試用系統 Gemini canvas
